# Sloomoo

**Sloomoo** is a creative AI-powered application built for a holiday party to bring joy and interactive experiences to attendees. It combines storytelling, visuals, and advanced AI to deliver a magical and memorable journey.

## Project Overview

Sloomoo invites users to:

1. **Experience a Poetic Journey**: Navigate through a poem paired with captivating videos.
2. **Make a Wish**: Submit a personalized wish at the end of the journey.
3. **Receive AI-Generated Surprises**:
   - A **custom picture** generated using a ComfyUI flux prompt (created by ChatGPT).
   - A **personalized poem** crafted by ChatGPT.

This unique blend of AI and creativity delivers a festive, engaging experience for all users.

## How to Run the Project

### Navigate to the Client and Server Directories:
- The **client** directory houses the frontend application.
- The **server** directory contains the backend logic.

### Follow Setup Instructions:
Each directory includes a detailed README for setup:
- [Client README](./client/README.md)
- [Server README](./server/README.md)

### Run the Application:
Once the client and server are running, visit [http://localhost:3000](http://localhost:3000) to begin your Sloomoo journey.

## Features

- **Interactive Poetry Experience**: Immerse yourself in a poetic narrative enhanced with videos.
- **AI-Generated Creativity**:
  - Personalized pictures created by a ComfyUI flux prompt.
  - Unique poems tailored to user wishes using ChatGPT.
- **Seamless Integration**: Real-time interactivity and AI responses for a smooth user experience.


## Technology Stack

- **Frontend**: React
- **Backend**: Express and Node.js
- **Real-Time Communication**: Socket.IO
- **AI Integration**: ChatGPT API
- **Visual Generation**: ComfyUI
- **Cloud Storage**: AWS S3 Bucket
