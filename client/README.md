# Sloomoo - Client

This directory contains the **frontend application** for Sloomoo, an AI-powered project designed to create an interactive poetic journey for users. The client is built with React and communicates with the backend server to provide a seamless and dynamic user experience.

---

## Features

- **Interactive User Interface**: Navigate through a responsive design paired with poems and videos.
- **AI-Driven Functionality**: Fetch personalized poems and images generated by the backend.
- **Real-Time Updates**: Powered by Socket.IO for instant communication.
- **User-Friendly Design**: Styled using Bootstrap and React-Bootstrap for a modern and clean look.


## Prerequisites

Make sure the following tools are installed:

- **Node.js** (version X.X or higher)
- **npm** or **yarn**


## Installation and Setup

1. Navigate to the `client` directory:
  
2. Install the required dependencies:
    ```bash
    npm install
    ```
3. Create an .env file in the client directory and add the following variable:
    ```bash
    REACT_APP_API_URL=http://localhost:3001
    ```
    Replace http://localhost:3001 with your backend server's URL if it’s different.

4. Start the development server:
    ```bash
    npm start
    ```
5. Open http://localhost:3000 in your browser to access the application.

## Scripts
The following scripts are available:

- `npm start`: Runs the application in development mode.
- `npm run build`: Builds the application for production.
- `npm test`: Launches the test runner in interactive mode.
- `npm run eject`: Ejects the application configuration (use with caution).

## Dependencies
The client application uses the following major dependencies:

- React: Framework for building user interfaces.
- React Router DOM: Enables navigation between pages.
- Socket.IO Client: Handles real-time communication.
- Axios: Manages API requests.
- Bootstrap & React-Bootstrap: Provides modern styling components.
- uuid: Generates unique IDs.
- bad-words: Filters offensive language.

Testing dependencies:
- @testing-library/jest-dom
- @testing-library/react
- @testing-library/user-event
